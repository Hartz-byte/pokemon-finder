## Getting Started

### Prerequisites

- Node.js 14.x or higher
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/pokemon-finder.git
   cd pokemon-finder
   ```
2. Install dependencies:
    ```bash
   npm install
    # or
    yarn install
   ```
3. Start the development server:
    ```bash
   npm run dev
    # or
    yarn dev
   ```
